# RVIDEO
RVIDEO is a video sharing website

## CONFIG
The .config file needs to be placed to the [cwd](https://en.wikipedia.org/wiki/Working_directory) 

Syntax for config file:
```
[ example .config file ]

[ makes sure to leave comments like this supports single and multi line comments ]

[SECRET_KEY <my-key>]

[PORT 
3000
]
```
